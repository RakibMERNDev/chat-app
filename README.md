
# Chat App

This is a basic chat app used with React, Nodejs, Socket io and Mongodb. It's a simple app to test socket io and it's implementation. 



## Features

- 32 themes selection thanks to Daisy Ui
- Realtime Chat Update
- Online/Offiline toggle to see online friends
- Image upload on chat and on profile section
- A decent minimalistic ui.
